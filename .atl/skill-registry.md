## Project Conventions

- **Clean Architecture**: Strict separation into Domain, Application, Infrastructure, and Presentation layers.
- **Strict TDD**: All features must be developed following the Red-Green-Refactor cycle.
- **Zod Validation**: Mandatory schema validation for all API inputs.

- **Database Patterns**: ACID Transactions, Optimistic Locking, and Cursor Pagination.
- **Event-Driven**: Use of Domain Events for asynchronous side effects.
